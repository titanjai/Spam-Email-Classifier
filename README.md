# Spam Email Classifier using Machine Learning

## Project Overview

This project classifies SMS messages as Spam or Ham using Machine Learning.

The model uses TF-IDF for feature extraction and Logistic Regression for classification.

## Technologies Used

- Python
- Pandas
- NLTK
- Scikit-Learn
- TF-IDF Vectorization
- Logistic Regression

## Dataset

SMS Spam Collection Dataset

Total Messages: 5572

## Data Preprocessing

- Lowercase Conversion
- Punctuation Removal
- Tokenization
- Stopword Removal
- Label Encoding

## Model Performance

Accuracy: 94.26%

## Features

- Detects Spam Messages
- Detects Ham Messages
- Custom Message Prediction

## Example

Input:

Free entry in a weekly competition. Win cash now!

Prediction:

Spam
